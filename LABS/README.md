# Labs

## Overview

This section documents the lab environments I use to practice cybersecurity skills in a safe, authorized, and educational setting.

My lab work is focused on building hands-on experience with Linux administration, system hardening, network scanning, Bash scripting, and technical documentation.

## Lab Goals

The purpose of my lab environment is to:

- Practice cybersecurity concepts legally and safely
- Build confidence with Linux and terminal workflows
- Test hardening steps and basic security configurations
- Run network discovery and enumeration against authorized systems
- Document repeatable processes and lessons learned

## Types of Lab Activities

In this lab, I use authorized systems to practice:

- Kali Linux setup and maintenance
- System updates and hardening
- Network scanning and host discovery
- Service enumeration
- Basic scripting and automation
- Security note-taking and project documentation

## Professional Standards

All activity documented in this portfolio is limited to:

- personal lab systems
- training environments
- authorized targets only

No unauthorized testing, real-world targeting, or sensitive data is included in this repository.
